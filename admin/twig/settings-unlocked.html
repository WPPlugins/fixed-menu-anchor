Ignore click event for certain CSS class

Here you can save the name of a certain CSS class, so if one anchor-link is clicked and it has this CSS-class in use, the click event, to move to the anchor target, is ignored. That is helpful in scenario where you use tabs or accordions, which actually need anchors to navigate. Leave empty, if you dont want to use it.
